Step 4 The Brain: ML Models

We implemented 4 models in src/models/ to find the best performer.

1

Why did it win? It accounts for User Bias.
Scenario: User A always rates 5 stars. User B always rates 1 star.
Result: KNNWithMeans subtracts the user's average rating to normalize these tendencies.

Located in src/inference/search_recommendation.py. This is how the real world interacts with the model.

The Cold Start Problem: What if a new product has no ratings?
Solution: We built a Hybrid System.
1. Try KNN (Collaborative Filtering).
2. If no results, Fallback to TF-IDF (Content-Based) using product names/categories.
def hybrid_recommend(product_id): knn_recs = knn_recommend(product_id) if len(knn_recs) >= 3: return knn_recs # High confidence return tfidf_recommend(product_id) # Fallback to Text matching
